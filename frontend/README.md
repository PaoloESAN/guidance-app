# Guidance App - Frontend

Frontend de la aplicación de orientación vocacional construida con Next.js, Ant Design y Supabase.

> **Nota:** Es posible que no funcione correctamente. Para la experiencia completa, ejecuta `wails dev` desde la carpeta raíz del proyecto.

## Configuración

1. Completa las variables en `.env`:
```
NEXT_PUBLIC_SUPABASE_URL="tu url de supabase"
NEXT_PUBLIC_SUPABASE_ANON_KEY="tu clave de supabase"
```

2. Instala dependencias:
```bash
npm install
```

3. Ejecuta el servidor de desarrollo:
```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
